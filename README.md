# ⭐ Featured Projects

## 🔐 Ultra-Low Power PQC Wearable Firmware

Developing an ultra-low power wearable coprocessor using **Zephyr RTOS**, **BLE**, **Embedded C**, and modular firmware architecture to support future FPGA-accelerated Post-Quantum Cryptography.

---

## 🛡 Post-Quantum DTLS Simulation Framework

LiteX-based simulation framework for evaluating secure communication using **ML-KEM (Kyber)**, **CRYSTALS-Dilithium**, **wolfSSL**, and FPGA-oriented hardware-software co-design.

---

## 🛰 Q-LEOGuard

Hybrid Quantum-Classical framework for **Satellite Collision Avoidance and Orbital Traffic Optimization**, integrating orbital mechanics, conjunction detection, and future **QUBO/QAOA-based optimization**.

---

## ⛓ Post-Quantum Secure Blockchain

Blockchain prototype integrating **CRYSTALS-Dilithium** digital signatures for quantum-resistant authentication and secure distributed systems.
